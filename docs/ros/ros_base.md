# ROS相关知识点
## 右手坐标系

X轴：正方向指向前方。X轴的正方向是从机器人底部（或基础）指向机器人前方。

Y轴：正方向指向机器人的左侧。Y轴的正方向是从机器人底部（或基础）指向机器人的左侧。

Z轴：正方向指向上方。Z轴的正方向是从机器人底部（或基础）指向机器人的上方。
